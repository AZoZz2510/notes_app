# Notes App 📝

A clean and professional Notes application built with **Flutter**, following **Clean Architecture** principles and using **Bloc/Cubit** for state management.

## ✨ Features
* **CRUD Operations:** Create, Read, Update, and Delete notes easily.
* **Smart Search:** Search notes instantly by their titles.
* **Color Customization:** Assign different colors to your notes for better organization.
* **Local Storage:** Your notes are saved locally using **Hive** database (works offline).
* **Responsive UI:** Clean and modern design implemented from Figma.

## 🛠 Tech Stack & Tools
* **Framework:** [Flutter](https://flutter.dev)
* **State Management:** [Cubit (flutter_bloc)](https://pub.dev/packages/flutter_bloc)
* **Local Database:** [Hive](https://pub.dev/packages/hive)
* **Architecture:** Clean Architecture (Presentation, Data, Domain layers).
